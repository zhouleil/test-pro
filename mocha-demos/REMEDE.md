
## How to use

First, clone the repo

```bash

```

Then, install the dependencies locally and Mocha globally.

```bash
$ cd mocha-demos
$ npm install
$ npm install --global mocha
```

Enter the `demo01` subdirectory.

```bash
$ mocha add.test.js
```

Now, play with the other examples under the repo's demo* directories.
